# ITAR/EAR Compliance for CNC Machining

## Overview
This document outlines export control requirements for CNC machining operations under ITAR (International Traffic in Arms Regulations) and EAR (Export Administration Regulations).

## ITAR (22 CFR Parts 120-130)

### Applicability to CNC Machining
- Applies to manufacturing of defense articles and technical data for defense services
- Covers CNC components for military applications (USML Categories III-VII, XV, XIX)

### Key Requirements
- **Registration**: Companies manufacturing USML items must register with DDTC
- **Deemed Exports**: Sharing technical data with foreign persons requires export license
- **Technical Data**: Includes manufacturing drawings, specifications, tolerances, processes
- **Specially Designed**: Manufacturing equipment/processes designed for military use

### CNC-Specific Controls
- Precision machining for aerospace/defense components
- Tool paths and G-code for controlled parts
- Material specifications for military applications
- Quality control procedures for defense contracts

## EAR (15 CFR Parts 730-774)

### Commerce Control List (CCL)
- **2B232**: Machine tools with advanced precision capabilities
- **3A611**: Military-specific integrated circuits
- **3B611**: Military test/inspection equipment
- **1C608**: Controlled chemicals/materials

### Export Restrictions
- Denied parties and countries (China, Russia, Iran, etc.)
- Re-export controls for items with controlled inputs
- Temporary import requirements for test articles

### Compliance Procedures
- Self-classification of items
- Commodity Jurisdiction requests for unclear cases
- Export Compliance Program documentation
- Record retention for 5 years

## Risk Mitigation
- Customer screening for foreign person status
- Technical data sanitization before sharing
- License verification for international shipments
- Documentation of export compliance decisions
