Agent Detail

Deep operational intelligence for any single agent.

Granular per-agent data: contract compliance, ARI breakdown, containment history, and real-time ingestion status. Everything an engineer needs.

Primary: ML Engineers / Ops

What this delivers

Operational command center for individual agents.

Engineers need granular per-agent data: contract compliance status, ARI risk breakdown, and containment action history. This dashboard delivers the deep intelligence required for root cause analysis and operational decisions.

📊

Contract Compliance Matrix

Real-time visibility into which of the 7 data contracts pass or fail, with specific validation errors for rapid remediation.

🎯

ARI Dimension Breakdown

Granular risk scoring across Autonomy, Impact, and Opacity dimensions to pinpoint exactly what's driving risk elevation.

📡

Tool-Agnostic Ingestion

Universal data ingestion via OTLP, Webhook, or Bulk JSON—works with LangSmith, Langfuse, MLflow, or any monitoring tool.

🛡️

MI9 Containment Timeline

Complete escalation history: what triggered each containment level, when it occurred, and what action was taken.

When you need this

Use Cases

🔍 Root Cause Analysis

Agent ARI spiked to 0.81. Drill into the 3 ARI dimensions (Autonomy, Impact, Opacity) to find the exact driver of the risk increase.

🔧 Contract Gap Remediation

C5 (Drift) and C7 (Decisions) are failing. View specific validation errors, identify the data pipeline issue, and fix it.

🛡 Containment Review

Agent at L3 containment. Review the full history: L0→L1 (drift detected) → L2 (guardrail breach) → L3 (manual escalation). Understand the journey.

🚧 Guardrail Event Investigation

Real-time guardrail event stream for this agent: what was blocked, when, and by which rule. Essential for safety debugging.

📡 Ingestion Health Check

Verify data is flowing from the team's chosen tool (LangSmith, Langfuse, etc.) via the configured protocol. Ensure telemetry is live.
